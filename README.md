# E-CELL CGC — Temporary Landing Website

A premium temporary launch website for E-CELL CGC built using React, Vite, Tailwind CSS, and Framer Motion. Designed as a professional holding page while the complete E-CELL platform is under development.

## Features

- Premium dark charcoal aesthetic
- Metallic typography effects
- Smooth Framer Motion animations
- Responsive Tailwind CSS design
- Interactive About page
- Mission, Vision and ecosystem sections
- Livestream route via Vercel rewrites
- E-Summit'26 navigation
- Vercel deployment ready

## Routes

- `/` → Landing page
- `/about` → About E-CELL
- `/livestream` → Rewritten to livestream deployment
- `esummit.ecellcgc.in` → E-Summit website

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
